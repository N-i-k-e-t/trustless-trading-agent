About This Project

Trustless Trading Agent

An autonomous AI trading agent that combines ERC-8004 on-chain identity with Kraken CLI execution to create a fully transparent, auditable, and trustless trading system.

Every trade decision is cryptographically signed on-chain before execution, creating an immutable audit trail that anyone can verify. The agent operates within strict risk guardrails enforced by smart contracts.
